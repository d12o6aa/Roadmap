📅 خطة زمنية مقترحة
12 شهر من الصفر للخبرة المتقدمة — بمعدل 10-15 ساعة أسبوعياً
شهر 1-3 — Foundation Phase
بناء الأساسيات
تعلم ML/LLM fundamentals، AI safety basics، security concepts. اقرأ 10+ أوراق بحثية أساسية وابنِ أول tool بسيطة.
ML Fundamentals AI Safety Security Concepts
شهر 4-7 — Core Phase
تعلم الهجمات الجوهرية
Prompt injection، jailbreaking، adversarial examples، data poisoning. ابنِ automated red teaming tool وانشر أول GitHub project.
Prompt Injection Jailbreaking Adversarial Data Poisoning
شهر 8-10 — Advanced Phase
التقنيات المتقدمة
Mechanistic interpretability، LLM agents security، multimodal attacks، privacy. شارك في competition وابدأ في كتابة research.
Interpretability Agent Security Multimodal Privacy
شهر 11-12 — Expert Phase
مستوى الخبراء والنشر
Automated red teaming research، safety benchmarking، defenses. انشر paper على arXiv أو في workshop وابنِ comprehensive platform.
